# E8 普遍クラス

## 物理的背景

臨界 Ising 場の理論（[Ising CFT](IsingCFT.md)）を縦磁場 σ で摂動すると、
質量ゼロのモードが 8 つの安定粒子に分裂する。
この質量スペクトルは E8 例外リー代数の基本表現の次元と一致する。

TFIM での対応: TFIM の臨界点 h=J に縦磁場 λ を加えた

```
H = -J Σ σᶻᵢ σᶻᵢ₊₁ - J Σ σˣᵢ - λ Σ σᶻᵢ
```

の低エネルギー有効理論が E8 粒子を持つ。

## 質量比（Zamolodchikov, 1989）

黄金比 ϕ = 2 cos(π/5) ≈ 1.6180... を用いると:

```
m₁ = 1
m₂ = ϕ                        ≈ 1.6180
m₃ = 2 cos(π/30)              ≈ 1.9890
m₄ = 2ϕ cos(7π/30)            ≈ 2.4049
m₅ = 2ϕ cos(2π/15)            ≈ 2.9563
m₆ = 2ϕ cos(π/30)             ≈ 3.2183
m₇ = 4ϕ² cos(7π/30)           ≈ 3.8912
m₈ = 4ϕ² cos(2π/15)           ≈ 4.7834
```

m₂/m₁ = ϕ（黄金比）は E8 の指標的な関係。

## 呼び出し例

```julia
ratios = fetch(:E8, :mass_ratios)
# → [1.0, 1.6180..., 1.9890..., 2.4049..., 2.9563..., 3.2183..., 3.8912..., 4.7834...]

# エイリアスも使える
fetch(:E8, :mass_ratio)
fetch(:E8, :E8_masses)
```

## 数値実験との比較時の注意点

- QAtlas が返すのは**場の理論の予言値**（熱力学極限・純粋量子状態）
- 有限サイズ格子計算との比較では有限サイズ補正が必要
- 実験値との比較では温度・磁場スケールの変換係数が必要
